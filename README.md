SmartRep is a simple, low-cost IoT system built on a Raspberry Pi 4B that solves this problem by monitoring gym
equipment in real time. Sensors are placed on or near each piece of equipment to detect whether it is currently in
use or available. This occupancy status, along with session duration, is logged and displayed on a live dashboard that
anyone can check before heading to the gym.

The system is straightforward by design: detect if equipment is being used, track how long each session lasts, and
display that information clearly. The focus is on reliable sensing, accurate logging, and a clean, readable dashboard.
